# 🌍 Azure Translator Web App

A simple and interactive **web-based translator** built using **Azure Cognitive Services (Translator API)**.  
This app allows users to translate text or files between multiple languages with a clean UI and flag-based selection.

---

## ✨ Features

- 🔑 Enter your own Azure API Key & Region  
- ✍️ Translate text using input textbox  
- 📂 Upload file and translate content  
- 🌍 Language selection with flags 🇮🇳 🇺🇸 🇫🇷 🇯🇵  
- ⚡ Fast and real-time translation  
- 🎨 Modern glassmorphism UI  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- Azure Translator API
